# setuq

## Agent skills

### Issue tracker

Issues tracked in GitHub Issues on `setuq-in/setuq` via the `gh` CLI. External PRs are not a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Default vocabulary: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Multi-context: `CONTEXT-MAP.md` at root points to per-context `CONTEXT.md` files (`engine/`, `ui/`, `cli/`). See `docs/agents/domain.md`.
